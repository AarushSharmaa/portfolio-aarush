# aarush-sharma

My personal portfolio. Built and maintained by me, for anyone who somehow ended up here.

Live at **[aarushsharmaa.github.io/aarush-sharma](https://aarushsharmaa.github.io/aarush-sharma/)**.

---

## what's inside

```
index.html      — about me, experience, the usual
work.html       — professional projects (enterprise GenAI, real clients)
projects.html   — lab work (open source, experiments, things I built for fun)
styles.css      — one stylesheet. no frameworks. no build step. just css.
```

That's it. No Node, no bundler, no `npm install`. Open `index.html` in a browser and it works.

---

## the stack

Pure HTML, CSS, and a few lines of vanilla JS. Deployed via GitHub Pages - every push to `main` goes live automatically.

Intentional choice. A portfolio doesn't need React.

---

## if you want to poke around

Clone it, open the files, change whatever. The CSS uses custom properties at the top of `styles.css` so colours are easy to swap. The structure is straightforward enough that you won't need a map.

If something's broken or looks off, feel free to open an issue. I'll get to it when I get to it.
