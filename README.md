
# Aarush Sharma - Portfolio

A clean, minimal portfolio website built with pure HTML and CSS. Inspired by modern design principles with focus on readability and performance.

## 🚀 Quick Start

1. Download all files and place them in a folder
2. Open `index.html` in your browser
3. That's it! Your portfolio is ready

## 📁 File Structure

```
portfolio/
├── index.html          # Main homepage
├── projects.html       # Projects showcase
├── blogs.html          # Blog/writing section
├── style.css           # Main stylesheet
└── README.md           # This file
```

## ✨ Features

- **Clean & Minimal Design**: Inspired by professional portfolio aesthetics
- **Fully Responsive**: Looks great on all devices
- **Fast Loading**: No heavy frameworks or images
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Easy to Customize**: Simple HTML/CSS structure

## 🎨 Customization

### Update Content
- Edit the text in each HTML file
- Replace placeholder links with your actual URLs
- Add your real GitHub/LinkedIn/Twitter profiles

### Colors
The main accent color is `#0066cc`. To change it:
1. Open `style.css`
2. Find and replace `#0066cc` with your preferred color

### Add More Pages
1. Create new HTML file (e.g., `about.html`)
2. Copy the structure from existing pages
3. Add navigation link in all files

## 🔧 Deployment

### GitHub Pages
1. Create a new repository on GitHub
2. Upload all files
3. Go to Settings → Pages
4. Select "Deploy from main branch"
5. Your site will be live at `username.github.io/repository-name`

### Netlify
1. Drag and drop the folder to Netlify
2. Your site will be live instantly

### Other Hosting
Upload all files to your web hosting service's public folder.

## 📝 Content Updates

### Projects
Edit `projects.html` to add/remove projects. Each project should have:
- Title and date
- Description
- Tech stack
- Links to demo/code

### Blog Posts
Edit `blogs.html` to add new articles. Update the "Recent Writing" section in `index.html` to match.

### Experience
Update the experience section in `index.html` with your work history.

## 🌟 Tips

- Keep the design minimal and content-focused
- Update regularly with new projects and posts
- Ensure all links work before publishing
- Test on mobile devices
- Use high-quality, compressed images if you add any
