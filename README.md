# aarush-sharma

My personal portfolio. Not because I think I'm a 10x engineer - mostly because I do too many things and this is the one place I can point at and say "here, this explains it."

Live at **[aarushsharmaa.github.io/aarush-sharma](https://aarushsharmaa.github.io/aarush-sharma/)**.

---

## what's inside

```
index.html      - about me, experience, the usual
work.html       - professional projects (enterprise GenAI, real clients)
projects.html   - lab work (open source, experiments, things I built because I could)
styles.css      - one stylesheet. no frameworks. no build step. just css.
```

No Node. No bundler. No `npm install`. Open `index.html` in a browser and it works. Tokens are unlimited but my patience for tooling is not.

---

## the stack

Pure HTML and CSS. Nothing else. Deployed via GitHub Pages, every push to `main` goes live.

A portfolio doesn't need React. I said what I said.

---

## if you want to poke around

Clone it, open the files, change whatever. CSS custom properties are at the top of `styles.css` so swapping colours takes 30 seconds. The structure is readable enough that you won't need a guide.

If something's broken, open an issue. I'll get to it between client calls.
