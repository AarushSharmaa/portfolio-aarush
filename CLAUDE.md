# Portfolio - Aarush Sharma

Static personal site. GitHub Pages, auto-deploys on push to `main`.

## Stack
Pure HTML / CSS / JS. No build tools, no frameworks. One shared stylesheet: `styles.css`.

## Rules
- Never push to `main` without explicit approval — it goes live immediately.
- Use `preview-*.html` for anything under review. Inline `<style>` in previews is fine.
- Do not add frameworks, bundlers, or external dependencies.
- Do not change the overall design language without approval.
- Test mobile (640px breakpoint) on any layout change.
